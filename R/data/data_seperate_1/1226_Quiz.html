가독성을 높이기 위해 plot형태로 만든다.

corrplot(crime_cor,
         method = "color",        # 색깔로 표현
         type = 'lower',          # 왼쪽 아래 행렬만 표시
         order = 'hclust',        # 유사한 상관계수끼리 군집화
         addCoef.col = 'black',   # 상관계수 표시 색
         tl.col = 'black',        # 변수명 표시 색
         tl.srt = 45,             # 변수명 기울기 각도
         diag = F)                # 대각행렬(본인과의 상관계수) 제외

강력 범죄의 검거율은 폭력(1) > 살인(0.95) > 강간(0.92) > 절도(0.89) 순으로 높다. 이를 통해 폭력은 범죄 발생 시 검거율이 100%에 달하며, 절도의 경우 상대적으로 낮은 검거율을 기록한다. 절도와 폭력은 0.9의 상관관계를 가지며, 매우 높은 유관성을 가진다. 이를 통해 범죄 발생 시 절도와 폭력이 함께 발생할 가능성이 높다고 유추할 수 있다.
